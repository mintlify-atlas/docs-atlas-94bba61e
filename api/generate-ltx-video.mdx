## Overview

The `generate_ltx_video.py` script provides inference for LTX-Video models, supporting both text-to-video and image-to-video generation. LTX-Video uses a multi-scale pipeline with advanced guidance techniques for high-quality video synthesis.

## Setup

Before running inference, convert PyTorch weights to JAX format:

```bash
cd src/maxdiffusion/models/ltx_video/utils
python convert_torch_weights_to_jax.py \
  --ckpt_path [LOCAL DIRECTORY FOR WEIGHTS] \
  --transformer_config_path ../ltxv-13B.json
```

## Usage

### Text-to-video

```bash
python src/maxdiffusion/generate_ltx_video.py \
  src/maxdiffusion/configs/ltx_video.yml \
  output_dir="[WEIGHTS DIRECTORY]" \
  config_path="src/maxdiffusion/models/ltx_video/ltxv-13B.json"
```

### Image-to-video

Add the conditioning image path to the config file:

```yaml
conditioning_media_paths: ["path/to/image.jpg"]
conditioning_start_frames: [0]
```

Then run:

```bash
python src/maxdiffusion/generate_ltx_video.py \
  src/maxdiffusion/configs/ltx_video.yml \
  output_dir="[WEIGHTS DIRECTORY]" \
  config_path="src/maxdiffusion/models/ltx_video/ltxv-13B.json"
```

## Output

Videos are saved in `outputs/YYYY-MM-DD/` directory with descriptive filenames based on the prompt:
- Videos: `video_output_{i}_{prompt_words}_{height}x{width}x{frames}_{index}.mp4`
- Images (single frame): `image_output_{i}_{prompt_words}_{height}x{width}x{frames}_{index}.png`

## Multi-scale pipeline

LTX-Video's multi-scale approach:

1. **First pass**: Generates video at lower resolution with dynamic guidance scaling across 7 stages
2. **Second pass**: Upscales spatially using a specialized upscaler model

This approach enables:
- Higher quality output through iterative refinement
- Better temporal consistency
- Efficient use of compute resources

## Advanced features

### Semantic Temporal Guidance (STG)

STG improves temporal coherence by applying guidance at different scales throughout the denoising process. Configure via `stg_scale` arrays in first/second pass configs.

### Block skipping

Selectively skip transformer blocks at different stages to optimize speed/quality tradeoff. Configure via `skip_block_list` arrays.

### CFG* rescaling

Advanced classifier-free guidance rescaling technique. Enable with `cfg_star_rescale: True`.

## Implementation details

LTX-Video features:
- 13B parameter transformer model
- T5-XXL text encoder
- 3D VAE for video compression
- Multi-scale generation pipeline
- Advanced guidance scheduling
- Optional prompt enhancement with Florence-2 and Llama
- Gaussian blur and CRF compression preprocessing
